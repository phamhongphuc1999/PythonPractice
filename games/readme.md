## Train

-   To train, run the command line like that

```shell
python3 train.py -n caro_v1 --game 1
```

## API

-   To start fastapi, run command line:

```shell
make run
```

-   And then you can open port 8000 to run api (docs link in http://127.0.0.1:8000/docs)
